# /brandforge index - Navigation & Discovery

## Command Structure
```yaml
---
command: "/brandforge index"
category: "Brand Navigation & Discovery"
purpose: "Comprehensive brand command catalog and project navigation"
wave-enabled: false
performance-profile: "standard"
---
```

## Purpose
Master navigation and discovery system for BrandForge commands, project structure, and brand assets. Provides intelligent command discovery, project exploration, and resource navigation.

## Usage Patterns
```bash
/brandforge index [query] --category [type]
/brandforge index --commands --examples --usage
/brandforge index --project --structure --navigation
/brandforge index --search [keyword] --filter [criteria]
```

## Arguments
- **query**: Search query or discovery topic
- **category**: Command category or type to explore

## Discovery Categories
- **commands**: Complete command catalog and usage
- **examples**: Practical examples and use cases
- **structure**: Project structure and organization
- **navigation**: Intelligent project navigation
- **templates**: Available templates and frameworks
- **workflows**: Common workflow patterns
- **glossary**: Brand strategy terminology and concepts

## Flags
- `--commands`: Complete BrandForge command catalog

## Quick Reference

### Getting Started
```bash
# Initialize new brand project
/brandforge init "My Brand" --discovery --strategy

# Complete brand development
/brandforge forge --complete --validate

# Strategic analysis
/brandforge analyze --comprehensive --competitive
```

### Common Workflows
```bash
# Research → Strategy → Creative → Launch
/brandforge research --comprehensive → strategy → creative → launch

# Validation-driven development
/brandforge validate --market-testing → improve → validate --performance

# Iterative enhancement
/brandforge improve --iterative --continuous --systematic
```

### Advanced Usage
```bash
# Multi-wave orchestration
/brandforge forge --wave-mode progressive --iterate --optimize

# Enterprise-scale development
/brandforge strategy --enterprise --comprehensive --validate

# Competitive intelligence
/brandforge analyze --competitive --benchmarking --insights
```
